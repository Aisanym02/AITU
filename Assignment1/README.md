
# Описание программы
Данная программа написана на языке C++ и демонстрирует использование параллельного программирования с помощью OpenMP.
В работе рассматриваются операции с динамическими массивами, поиск минимума и максимума, вычисление среднего значения, а также сравнение последовательных и параллельных алгоритмов по времени выполнения.
Программа состоит из четырёх заданий, каждое из которых направлено на изучение разных аспектов работы с массивами и параллельными вычислениями.
# Цель работы
Целью данной работы является:
изучение принципов работы OpenMP;
освоение директив parallel for и reduction;
сравнение производительности последовательных и параллельных вычислений;
практика работы с динамической памятью в C++.
# Используемые технологии
Язык программирования: C++
Параллельное программирование: OpenMP
Генерация случайных чисел: rand(), srand()
Измерение времени выполнения: библиотека <chrono>
Работа с динамической памятью: new[] и delete[]
# Структура программы
## Задание 1 — Вычисление среднего значения массива
Создаётся динамический массив из 50 000 целых чисел.
Массив заполняется случайными значениями от 1 до 100.
Последовательно вычисляется сумма и среднее значение элементов массива.
Память после использования освобождается.
## Задание 2 — Последовательный поиск минимума и максимума
Создаётся динамический массив из 1 000 000 элементов.
Выполняется последовательный поиск минимального и максимального значения.
Измеряется и выводится время выполнения алгоритма.
## Задание 3 — Параллельный поиск минимума и максимума (OpenMP)
Используется тот же массив из 1 000 000 элементов.
Поиск минимума и максимума выполняется параллельно с помощью OpenMP.
Применяется директива reduction(min: ...) и reduction(max: ...).
Результаты и время выполнения сравниваются с последовательной версией.
## Задание 4 — Последовательное и параллельное вычисление среднего
Создаётся массив из 5 000 000 элементов.
Последовательно вычисляется среднее значение элементов массива.
Затем то же вычисление выполняется параллельно с использованием reduction(+:sum).
Сравнивается время выполнения обоих вариантов.
Компиляция и запуск
Для компиляции программы требуется компилятор с поддержкой OpenMP.
Пример компиляции с использованием g++:
g++ -O2 -fopenmp main.cpp -o main
Запуск программы:
./main
# Выводы
Параллельные вычисления позволяют значительно ускорить операции на больших массивах.
Использование reduction в OpenMP обеспечивает корректное и эффективное объединение результатов.
Для небольших массивов выигрыш от параллельности может быть незначительным.
Корректное управление динамической памятью является важной частью программирования на C++.
