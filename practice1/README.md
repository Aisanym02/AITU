# Описание задания
Данная программа является учебной работой по параллельному программированию с использованием OpenMP на языке C++.
В работе рассматриваются базовые операции с массивами, структурами данных и динамической памятью, а также сравнивается производительность последовательных и параллельных реализаций.
Программа разделена на три логические части, каждая из которых демонстрирует разные аспекты работы с данными и параллельными вычислениями.
# Цель работы
Целью данной работы является:
изучение принципов параллельного программирования с использованием OpenMP;
сравнение последовательных и параллельных вычислений;
демонстрация влияния синхронизации и редукций на производительность;
практика работы с динамической памятью и указателями.
# Структура программы
## Часть 1 — Работа с массивами
В первой части программы:
создаётся массив случайных целых чисел;
выполняется поиск минимального и максимального элементов:
последовательно;
параллельно с использованием OpenMP.
Для параллельной версии используется подход с локальными минимумами и максимумами для каждого потока и их объединение через критическую секцию.
Также измеряется и выводится время выполнения каждого варианта.
## Часть 2 — Структуры данных
Во второй части реализованы собственные структуры данных:
односвязный список;
стек (на динамическом массиве);
очередь (кольцевой буфер).
Для списка и очереди дополнительно реализовано:
последовательное добавление элементов;
параллельное добавление элементов с использованием #pragma omp critical.
Проводится сравнение времени выполнения последовательного и параллельного добавления большого количества элементов, а также делается вывод о влиянии синхронизации на производительность.
## Часть 3 — Динамическая память и указатели
В третьей части:
создаётся динамический массив с помощью оператора new;
вычисляется среднее значение элементов массива:
последовательным способом;
параллельно с использованием директивы reduction;
измеряется время выполнения каждого варианта;
динамическая память корректно освобождается с помощью delete[].
Эта часть демонстрирует эффективное использование редукций в OpenMP.
