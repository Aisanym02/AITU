# Параллельные сортировки на CUDA

## Описание задания
Данный проект выполнен в рамках учебного задания по параллельному программированию и посвящён реализации и сравнению **параллельных алгоритмов сортировки на GPU с использованием CUDA**, а также их **последовательных аналогов на CPU**.

В работе рассматриваются следующие алгоритмы сортировки:
- сортировка слиянием (Merge Sort),
- быстрая сортировка (Quick Sort),
- пирамидальная сортировка (Heap Sort).

Для каждого алгоритма реализованы:
- последовательная версия на CPU;
- параллельная версия на GPU (CUDA).

---

## Цель работы
- Изучить принципы параллельного программирования на GPU.
- Реализовать параллельные алгоритмы сортировки с использованием CUDA.
- Сравнить производительность CPU и GPU реализаций.
- Проанализировать влияние размера данных на ускорение вычислений.

---

## Краткое описание реализованных алгоритмов

### 1️⃣ Параллельная сортировка слиянием (CUDA Merge Sort)
- Массив делится на подмассивы (блоки).
- Каждый блок потоков CUDA сортирует свой подмассив параллельно.
- Далее выполняются итеративные этапы параллельного слияния отсортированных подмассивов.
- Алгоритм хорошо масштабируется и наиболее эффективно использует архитектуру GPU.

### 2️⃣ Параллельная быстрая сортировка (CUDA Quick Sort)
- Массив делится на части (чанки).
- Каждая часть сортируется параллельно на GPU.
- После этого результаты объединяются.
- Данный подход соответствует учебной параллельной реализации быстрой сортировки на GPU.

### 3️⃣ Параллельная пирамидальная сортировка (CUDA Heap Sort)
- Классическая пирамидальная сортировка плохо распараллеливается из-за большого количества зависимостей.
- В рамках учебной работы используется параллельная блочная обработка данных с последующим объединением результатов.
- Реализация корректна и демонстрирует ограничения heap sort при переносе на GPU.
